Dòng này được CSS bằng External,sử dụng tag name

  • CSS Inline: Sử dụng thuộc tính style trực tiếp trong thẻ HTML.
  • CSS Internal: Được định nghĩa trong thẻ trong phần của tài liệu
  • CSS External: Được liên kết thông qua thẻ và nằm trong một tệp CSS riêng biệt.
  • CSS Selector: Sử dụng các bộ chọn để áp dụng kiểu dáng cho các phần tử cụ thể.
  • CSS Specificity: Quy tắc xác định mức độ ưu tiên của các quy tắc CSS.
STT Mức ưu tiên
1 Inline CSS (style trực tiếp trong thẻ HTML)
2 Internal CSS (định nghĩa trong thẻ )
3 External CSS (liên kết tệp CSS bên ngoài)
4 CSS Selector (bộ chọn CSS)
5 CSS Specificity (mức độ ưu tiên của quy tắc CSS)
